# 🪨📄✂️ Rock Paper Code

A sleek, draggable retro-style **Rock, Paper, Scissors** desktop widget built with **Electron**, custom **pixel art assets**, and **glassmorphism UI**.

---

##  Features

*  **Character Selection:** Choose your player avatar (Girl or Boy) before starting a match.
*  **Smart AI Opponent:** Play against a pixel robot enemy.
*  **Best of 3 Match Mode:** Track wins with interactive round indicators (`⭐`).
*  **Frameless & Floating:** Transparent, borderless window that floats above your desktop apps.
*  **Draggable UI:** Drag the card anywhere across your desktop screen.
*  **Pixel Art Aesthetic:** Styled with custom 16-bit graphics and retro arcade typography (`Press Start 2P`).

---

##  Built With

* **HTML5 / CSS3** (Glassmorphism UI, Flexbox, Animations)
* **JavaScript (ES6)** (Game state management & score tracking)
* **[Electron](https://www.electronjs.org/)** (Desktop application framework)
